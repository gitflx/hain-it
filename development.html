Reference Project

Patient Portal

Industry: Healthcare / Medical Center

Initial Situation

Patients had to call for appointment changes, lab results were sent by mail, forms were filled out on paper. High administrative overhead at the practice.

Solution

  • Self-service portal with online appointment booking
  • Digital forms with pre-population
  • Secure lab result retrieval (Azure AD B2C, end-to-end encrypted)
  • Integration with practice system via HL7 FHIR interface

Outcome

70% fewer phone inquiries. Average waiting time for result notification reduced from 5 days to under 24 hours.

Reference Project

IoT Monitoring Dashboard

Industry: Renewable Energy

Initial Situation

An operator of 45 solar installations had no centralized overview. Failures were only noticed through missing feed-in tariff payments — often days later.

Solution

  • Real-time dashboard with live data from all installations (React + SignalR)
  • Automatic anomaly detection and alerting
  • Historical analysis and performance comparison
  • Mobile-optimized view for on-site technicians

Outcome

Mean response time for failures reduced from 4 days to 35 minutes. Revenue increase of 12% through faster issue resolution.

Reference Project

Omnichannel API

Industry: E-Commerce / Retail

Initial Situation

Inventory, prices, and orders had to be manually synchronized between the webshop (Shopify), marketplaces (Amazon, eBay), and POS system. High error rate, frequent overselling.

Solution

  • Event-driven API layer with Azure Service Bus
  • Real-time inventory synchronization across all channels
  • Centralized pricing logic with channel-specific rules
  • Retry mechanism and dead-letter queue for error handling

Outcome

Overselling rate reduced from 8% to under 0.2%. Manual synchronization eliminated. Scaled to 3 additional marketplaces without extra effort.
