Reference project

M365 Migration & Governance

Industry: Consulting firm, 200 employees

Background

On-premise Exchange 2016, unstructured file shares, no mobile device strategy. High administrative overhead, no centralized governance.

Solution

  • Phased migration to Exchange Online and SharePoint
  • Governance framework with naming conventions, retention policies, and DLP
  • Intune rollout for BYOD and corporate devices
  • Training concept with a champions network

Result

Full cloud migration completed in 4 months. IT operating costs reduced by 40%. User satisfaction significantly improved through self-service capabilities.

Reference project

Copilot rollout

Industry: Management consultancy, 80 employees

Background

Leadership wanted to deploy Microsoft 365 Copilot but was uncertain about readiness, data privacy, and expected ROI.

Solution

  • Readiness assessment: data quality, permission structure, oversharing check
  • Pilot group of 15 users across different departments
  • Time savings measured per use case (email, meetings, documents)
  • Adoption campaign with a prompt library and weekly tips

Result

Average time savings of 4.5 hours per user per week in the pilot group. Full rollout to all users after 6 weeks. Positive reception thanks to hands-on onboarding.

Reference project

Power Platform Center of Excellence

Industry: Manufacturing company, 500+ employees

Background

Scattered Power Apps across business units, no centralized governance. Shadow IT was growing unchecked, raising security concerns from the IT department.

Solution

  • Deployment of the Microsoft CoE Toolkit
  • Environment strategy (Dev/Test/Prod) with DLP policies
  • Template library and training program for citizen developers
  • Approval workflow for production apps

Result

Controlled scaling from 12 to 85 production apps in 12 months. Zero security incidents. IT department repositioned as an enabler rather than a blocker.
